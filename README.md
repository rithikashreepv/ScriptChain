# Learnable Positional Encoding (PyTorch)

This project implements learnable positional encoding using PyTorch.

## 🚀 Features
- Learnable positional encoding using nn.Embedding
- Simple neural network model
- Dummy dataset for training
- Training loop with loss reduction

## 🧠 Concept
Transformers do not understand sequence order. This project adds positional information using trainable embeddings.

## 🛠️ Tech Stack
- Python
- PyTorch

## 📌 Output
Model learns to predict values using positional information.
